# Northbridge Digital Banking

Northbridge's retail banking dashboard, built on Angular 14 and the Keystone Design System (`@northbridge/ui-components`).
The Accounts Dashboard shows balance summaries and recent transactions, filtered through Keystone components.
Run `nvm use 16 && npm ci && npm start`, then open http://localhost:4200/dashboard.
Tests: `npm test` — builds: `npm run build`.
